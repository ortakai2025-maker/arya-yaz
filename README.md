# 🎮 Aryasu - Yaz Macerası

Aryasu adlı bir çocuk için tasarlanan interaktif yaz aktivite takip uygulaması.

## ✨ Özellikler

- 📅 Günlük aktivite kontrolü
- 🎯 Haftalık planlama
- ♟️ Satranç bulmacaları
- 🧩 Sudoku oyunları
- 📚 İngilizce kelime quiz'leri
- 🇪🇸 İspanyolca öğrenme
- 🧮 Matematik testleri
- 🎸 Gitar eğitimi ve akord öğrenmesi
- 📖 Okuma takibi
- 🎬 Film/kitap/müzik günlüğü

## 🚀 Başlangıç

### Kurulum
```bash
npm install
```

### Geliştirme Sunucusu
```bash
npm run dev
```
Tarayıcıda `http://localhost:3011` adresine git.

### Üretim İçin Derle
```bash
npm run build
```

## 🔧 Teknoloji Stack

- **Framework:** React 18
- **Build Tool:** Vite 5
- **Styling:** Tailwind CSS 3
- **Audio:** Tone.js
- **Icons:** Lucide React
- **Deploy:** Netlify

## 📝 Ortam Değişkenleri

`.env` dosyası oluştur (`.env.example` dosyasını referans al):

```env
ANTHROPIC_API_KEY=sk-ant-your-key-here
```

## 🌐 Netlify Deploy

1. GitHub'da repo aç
2. Netlify'a bağla
3. Ortam değişkeni ayarla
4. Deploy et
